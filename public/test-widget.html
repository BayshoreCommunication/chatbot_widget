Chat Widget Test Page

Testing Features:

  • Auto-Open Widget: Widget should automatically open when page loads (if enabled in dashboard)
  • Video Autoplay: Video should automatically play when widget opens (if enabled in dashboard)
  • Manual Open: Click the chat button to manually open the widget
  • Video Controls: Video should stop after the configured duration

Instructions:

  • Make sure your backend server is running on localhost:8000
  • Configure auto-open and video settings in the dashboard
  • Refresh this page to test the widget behavior
  • Check browser console for debug logs

Debug Information:

Open browser console (F12) to see debug logs for:

  • Settings loading
  • Auto-open behavior
  • Video playback
  • API responses
